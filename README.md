# Spatial Application 
# How to run it on Windows

```
python -m venv venv
```

``` 
.\venv\Scripts\Activate.ps1
```

```
Set-ExecutionPolicy -Scope CurrentUser -ExecutionPolicy RemoteSigned
```

``` 
pip install -r requirements.txt
```

``` 
python run.py 
```

# How to run it on macOS

```
python3 -m venv venv
```

```
source venv/bin/activate
```

```
pip install -r requirements.txt
```

```
python run.py
```
